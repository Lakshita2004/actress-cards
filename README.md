Project Overview:

This project is a simple web page that displays profile cards of four popular Bollywood actresses. Each card includes an image, name, short description, skills, and a contact button.
The purpose of this project is to practice HTML structure and basic CSS styling while building a visually organized layout.

Features:
Profile cards for 4 actresses
Image, description, and skills section in each card
Styled buttons with basic interaction (alert message)
Clean and readable layout
Use of CSS properties like:
display: flex
border-radius
object-fit
margin and background-color

Technologies Used:
HTML5
CSS3
